Information about Images Tag.

Unlike most elements, the image tag is a specific reference in your HTML document to an image on your website.

It is more like an advanced link to an image file. The HTML image tag actually retrieves the image and displays it in your web page.

However, unlike most elements, the image tag requires an attribute in order to display an image.
