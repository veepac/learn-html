Information About table Tags.

The table tag defines an HTML table.

An HTML table consists of the table element and one or more tr tag, th tag, and td tag elements.

The tr tag element defines a table row, the th tag element defines a table header, and the td tag element defines a table cell.

A more complex HTML table may also include caption, col, colgroup>, thead, tfoot, and tbody elements.
